商品搜索链路
步骤实现
1. 分词ES IK analyzer（ik_max_word 索引 / ik_smart 搜索）
2. 查询multi_match（title 权重 x3 + description），fuzziness=AUTO
3. 过滤categoryId / type / condition / price 范围 / status=0
4. 高亮title 和 description 字段 <em> 标签
5. 回查ES 取 ID → MySQL selectBatchIds 按 ES 顺序返回
6. 降级ES 不可用时自动走 MySQL LIKE 查询
