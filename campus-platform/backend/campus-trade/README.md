# campus-trade

交易微服务，平台核心模块，涵盖商品发布、搜索、下单、聊天、配送、评价、纠纷等完整 C2C 交易链路。

**端口**: 8082 | **服务名**: `campus-trade-service`

## 功能

- **商品** — 发布/编辑/下架/刷新、多条件筛选、ES 全文搜索、个性化推荐、浏览历史
- **订单** — 自提/配送两种模式、核销码确认、订单状态流转、超时自动取消
- **实时聊天** — WebSocket 即时通讯、消息撤回（2分钟内）、防诈骗敏感词检测
- **配送** — 跑腿员抢单、取货/送达轨迹、实时位置上报（10秒限流）
- **评价** — 交易完成后互评、评分统计、24小时内可编辑/删除
- **社交** — 收藏商品、关注用户、拉黑用户
- **纠纷/举报** — 订单纠纷、内容举报
- **后台管理** — 商品审核、订单管理、纠纷处理、举报处理

## 订单状态流转

```
自提:  待确认(0) → 已确认(1) → 已完成(3)
                    ↘ 已取消(4)

配送:  待确认(0) → 配送协商(5) → 待派单(6) → 已接单(7) → 已取货(8) → 已送达(9) → 已完成(3)
                    ↘ 已取消(4)
```

## 特殊能力

- **Elasticsearch** — 商品全文搜索（IK 中文分词）
- **WebSocket** — 买卖双方实时聊天
- **Redisson** — 分布式锁（抢单并发控制）、限流（位置上报）
- **定时任务** — 超时订单自动取消、未刷新商品自动下架

## 依赖

campus-common、campus-feign、Elasticsearch Java Client
